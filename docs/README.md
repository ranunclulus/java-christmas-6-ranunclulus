# 🎄 크리스마스 이벤트 🎄

  - [x]  Enum 생성
      - [x]  메뉴 Enum
      - [x]  음식 종류 Enum
      - [x]  특별 할인 Enum

    - [ ]  혜택으로 인한 총 할인 금액 출력

    - [ ]  배지 계산 후 출력
